# Changelog

Все значительные изменения в проекте будут документированы в этом файле.

Формат основан на [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
и проект следует [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Полная настройка проекта с современным React стеком
- React 19 с TypeScript 5.7
- Vite 7 для быстрой разработки
- Tailwind CSS v4 с новым CSS-first подходом
- shadcn/ui компоненты для UI
- TanStack Router для типобезопасной маршрутизации
- TanStack Query для управления серверным состоянием
- Zustand для управления глобальным состоянием
- React Hook Form с Zod валидацией
- Biome для линтинга и форматирования
- Vitest и React Testing Library для тестирования
- Lefthook для Git hooks
- Commitlint для валидации коммитов
- GitHub Actions CI/CD workflows
- Comprehensive layout компоненты
- Публичные и приватные страницы
- Система аутентификации с route guards
- Полная документация

### Changed
- Обновлены все зависимости до последних версий

### Fixed
- Исправлены TypeScript ошибки в layout компонентах
- Исправлена интеграция MobileNav с Header

## [0.1.0] - 2024-01-XX

### Added
- Начальная настройка проекта
- Базовая структура папок
- Конфигурация TypeScript
- Конфигурация Vite
- Базовые компоненты UI

## Типы изменений

- `Added` - новая функциональность
- `Changed` - изменения в существующей функциональности
- `Deprecated` - функциональность, которая скоро будет удалена
- `Removed` - удаленная функциональность
- `Fixed` - исправления багов
- `Security` - исправления уязвимостей
