# Multi-Select Demo Page

Интерактивная демонстрация компонента Multi-Select.

## Доступ

Откройте в браузере: `http://localhost:5173/demo/multi-select`

Или перейдите с главной страницы по ссылке "View Multi-Select Demo".

## Что демонстрируется

### 1. Базовый пример
- Простой мультиселект с выбором фреймворков
- Отображение выбранных значений в JSON формате
- Удаление выбранных элементов через крестик

### 2. С группировкой
- Опции сгруппированы по категориям (Frontend, Backend, Database)
- Заголовки групп (MultiSelectLabel)
- Визуальное отображение выбранных элементов

### 3. Overflow Behavior
- Три режима отображения:
  - **wrap-when-open** - переносит только когда открыт
  - **wrap** - всегда переносит
  - **cutoff** - показывает первый + счетчик
- Интерактивное переключение режимов
- Предзаполненные значения через `defaultValues`

### 4. Контролируемое состояние
- Управление через внешние кнопки
- Кнопки: Сбросить, Выбрать все, Популярные
- Счетчик выбранных элементов

### 5. В форме
- Использование в HTML форме
- Валидация (обязательное поле)
- Отображение ошибок
- Успешная отправка с результатом

## Запуск

```bash
# Установка зависимостей
pnpm install

# Запуск dev сервера
pnpm run dev

# Откройте http://localhost:5173/demo/multi-select
```

## Код

Исходный код демо-страницы: `src/routes/demo/multi-select.tsx`

## Возможности для тестирования

1. **Выбор элементов** - кликайте на опции для выбора/отмены
2. **Удаление** - кликайте на крестик в badge для удаления
3. **Overflow** - переключайте режимы и смотрите как меняется отображение
4. **Группы** - выбирайте элементы из разных групп
5. **Форма** - попробуйте отправить пустую форму (валидация)
6. **Контроль** - используйте кнопки для управления выбором
