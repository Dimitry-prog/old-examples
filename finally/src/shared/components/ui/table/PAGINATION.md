# Расширенная пагинация

## Возможности

✅ **Кнопки навигации**
- Первая страница (⏮)
- Предыдущая страница (◀)
- Следующая страница (▶)
- Последняя страница (⏭)

✅ **Номера страниц**
- Отображение номеров страниц
- Троеточие (...) для пропущенных страниц
- Подсветка текущей страницы
- Можно скрыть через `showPageNumbers={false}`

✅ **Выбор количества строк**
- Dropdown с опциями (по умолчанию: 10, 20, 50, 100)
- Настраивается через `pageSizeOptions`

✅ **URL-based состояние**
- Состояние хранится в URL
- Можно делиться ссылкой
- Работает с кнопками браузера

## Пример

```tsx
<TableData
  name="products"
  columns={columns}
  fetchData={fetchProducts}
  pageSize={10}
  pageSizeOptions={[5, 10, 20, 50, 100]}
  showPageNumbers={true}  // Показывать номера страниц (по умолчанию true)
/>
```

## Визуальный пример

```
┌─────────────────────────────────────────────────────────────────┐
│ Rows per page: [10 ▼]    Page 3 of 10  ⏮ ◀ 1 2 [3] 4 5 ... 10 ▶ ⏭ │
└─────────────────────────────────────────────────────────────────┘
                                                        ↓ (клик)
                                                   ┌─────────┐
                                                   │ Page 6  │
                                                   │ Page 7  │
                                                   │ Page 8  │
                                                   │ Page 9  │
                                                   └─────────┘
```

## Как это работает

1. **Выбор количества строк** (слева)
   - Dropdown с опциями
   - При изменении сбрасывает на первую страницу
   - Обновляет URL параметр

2. **Информация о странице** (центр)
   - "Page 3 of 10"
   - Всегда видна

3. **Навигация** (справа)
   - ⏮ - Первая страница
   - ◀ - Предыдущая страница
   - Номера страниц (1, 2, [3], 4, 5, ..., 10)
   - ▶ - Следующая страница
   - ⏭ - Последняя страница

4. **Умное отображение номеров**
   - Если страниц ≤ 7: показываются все
   - Если страниц > 7: показываются с троеточием
   - Всегда видны: первая, последняя, текущая ±1

5. **Dropdown для троеточия**
   - Клик на ... открывает dropdown
   - Показывает скрытые страницы
   - Клик вне dropdown закрывает его
   - Выбор страницы из списка переходит на неё

## Настройки

### Скрыть номера страниц

```tsx
<TableData
  name="products"
  columns={columns}
  fetchData={fetchProducts}
  showPageNumbers={false}  // Только кнопки навигации
/>
```

Результат:
```
Rows per page: [10 ▼]    Page 3 of 10  ⏮ ◀ ▶ ⏭
```

### Кастомные опции размера

```tsx
<TableData
  name="products"
  columns={columns}
  fetchData={fetchProducts}
  pageSizeOptions={[25, 50, 100, 200]}  // Свои значения
/>
```

### Отключение пагинации

Пагинация автоматически отключается при:
- `isLoadOnScroll={true}` - используется бесконечная прокрутка
- `totalRows <= pageSize` - все данные на одной странице

## Использование отдельно

```tsx
import { TablePagination } from "@/shared/components/ui/table";

<TablePagination
  table={tableInstance}
  name="my-table"
  pageSizeOptions={[10, 20, 50]}
  showPageNumbers={true}
/>
```

## Стилизация

Компоненты:
- **Button** - для всех кнопок (outline/default варианты)
- **select** - для dropdown выбора размера
- **Иконки** - из lucide-react (ChevronsLeft, ChevronLeft, etc.)

Можно кастомизировать через Tailwind классы или глобальные стили.
