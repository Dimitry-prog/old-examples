# Визуальное руководство по пагинации

## Полная пагинация (по умолчанию)

```
┌──────────────────────────────────────────────────────────────────────┐
│                                                                      │
│  Rows per page: [10 ▼]      Page 3 of 10   ⏮ ◀ 1 2 [3] 4 5 ... 10 ▶ ⏭  │
│                                                                      │
└──────────────────────────────────────────────────────────────────────┘
   ↑                          ↑                ↑
   Выбор размера          Информация      Навигация с номерами
```

## Без номеров страниц (showPageNumbers={false})

```
┌──────────────────────────────────────────────────────────────────────┐
│                                                                      │
│  Rows per page: [10 ▼]      Page 3 of 10         ⏮ ◀ ▶ ⏭            │
│                                                                      │
└──────────────────────────────────────────────────────────────────────┘
   ↑                          ↑                    ↑
   Выбор размера          Информация      Только кнопки навигации
```

## Логика отображения номеров страниц

### Мало страниц (≤ 7)
```
Страница 1:  ⏮ ◀ [1] 2 3 4 5 ▶ ⏭
Страница 3:  ⏮ ◀ 1 2 [3] 4 5 ▶ ⏭
Страница 5:  ⏮ ◀ 1 2 3 4 [5] ▶ ⏭
```

### Много страниц (> 7)
```
Страница 1:   ⏮ ◀ [1] 2 3 ... 20 ▶ ⏭
                         ↓ (клик на ...)
                    ┌─────────┐
                    │ Page 4  │
                    │ Page 5  │
                    │ ...     │
                    │ Page 19 │
                    └─────────┘

Страница 2:   ⏮ ◀ 1 [2] 3 4 ... 20 ▶ ⏭
Страница 3:   ⏮ ◀ 1 2 [3] 4 ... 20 ▶ ⏭
Страница 10:  ⏮ ◀ 1 ... 9 [10] 11 ... 20 ▶ ⏭
                  ↓              ↓
            Dropdown 1      Dropdown 2
            (страницы       (страницы
             2-8)            12-19)

Страница 18:  ⏮ ◀ 1 ... 17 [18] 19 20 ▶ ⏭
Страница 19:  ⏮ ◀ 1 ... 18 [19] 20 ▶ ⏭
Страница 20:  ⏮ ◀ 1 ... 18 19 [20] ▶ ⏭
```

## Состояния кнопок

### Первая страница
```
⏮ ◀ [1] 2 3 ▶ ⏭
↑ ↑
Отключены (disabled)
```

### Последняя страница
```
⏮ ◀ 8 9 [10] ▶ ⏭
              ↑ ↑
              Отключены (disabled)
```

### Средняя страница
```
⏮ ◀ 4 [5] 6 ▶ ⏭
↑ ↑         ↑ ↑
Все активны
```

## Выбор количества строк

```
┌─────────────────────┐
│ Rows per page: [10 ▼]│
└─────────────────────┘
                    ↓
        ┌──────────────┐
        │ 10           │
        │ 20           │
        │ 50           │
        │ 100          │
        └──────────────┘
```

При выборе нового значения:
1. Обновляется URL: `?table=0-20`
2. Сбрасывается на первую страницу
3. Перезагружаются данные

## Цветовая схема

- **Текущая страница**: синий фон (variant="default")
- **Другие страницы**: белый фон с рамкой (variant="outline")
- **Отключенные кнопки**: серый цвет, opacity 50%
- **Троеточие**: серый цвет, не кликабельно

## Примеры использования

### Стандартная таблица
```tsx
<TableData
  name="users"
  columns={columns}
  fetchData={fetchUsers}
  pageSize={10}
  pageSizeOptions={[10, 20, 50, 100]}
/>
```

### Компактная (без номеров)
```tsx
<TableData
  name="logs"
  columns={columns}
  fetchData={fetchLogs}
  pageSize={50}
  showPageNumbers={false}
/>
```

### Большие данные
```tsx
<TableData
  name="transactions"
  columns={columns}
  fetchData={fetchTransactions}
  pageSize={100}
  pageSizeOptions={[50, 100, 200, 500]}
/>
```
